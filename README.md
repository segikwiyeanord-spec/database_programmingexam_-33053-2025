# database_programmingexam_-33053-2025
The Warehouse Management System is an Oracle PL/SQL database application designed to manage suppliers, products, customers, and orders. It automates inventory tracking, updates stock levels, processes sales transactions, and generates reports using tables, procedures, functions, anonymous blocks, and window functions.
The project demonstrates the use of advanced Oracle PL/SQL concepts such as tables, constraints, sequences, procedures, functions, triggers, cursors, packages, anonymous blocks, exception handling, and window functions to ensure efficient data processing and business automation.

Features
Supplier management (add, update, delete, and view suppliers)
Product management with stock quantity tracking
Customer registration and management
Order creation and sales transaction processing
Automatic inventory updates after each sale
Low-stock monitoring and inventory control
Data validation and error handling using PL/SQL exceptions
Business reports using SQL queries and window functions
Secure and consistent database operations through stored procedures and functions
